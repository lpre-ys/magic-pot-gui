Magic Pot GUI
==============

Magic Pot の Windows 向け GUI 版です。

------------------------------------------------------------
■ 必要環境
------------------------------------------------------------
本ツールは ImageMagick v7 に依存しています。
事前に ImageMagick をインストールし、
コマンドプロンプトから「magick」と入力して
動作することを確認してください。

ダウンロード:
  https://imagemagick.org/

------------------------------------------------------------
■ インストール方法
------------------------------------------------------------
- インストーラ版:
  インストーラを実行してください。

- ZIP版:
  解凍後、フォルダ内の「magic_pot.exe」を実行してください。

------------------------------------------------------------
■ 使い方
------------------------------------------------------------
1. 出力先フォルダを選択
   ツクールのプロジェクトの「Picture」フォルダを指定してください。

2. 画像を投入
   PNG ファイルをドラッグ＆ドロップするか、
   クリックしてファイル選択ダイアログから選んでください。
   一度に最大 1000 件まで投入できます。

3. ログを確認
   - 処理中は「砂時計」アイコンが表示されます。
   - 完了すると「チェックマーク」に変わります。
   - 出力先フォルダに画像が保存されます。

4. 失敗した場合
   - 成功数と失敗数が表示されます。
   - 失敗数にカーソルを合わせるとファイル名を確認できます。

5. ログを消したい場合
   - 「ゴミ箱」アイコンをクリックするとログが消えます。
   - 消えるのはログだけです。生成した画像は消えません。
